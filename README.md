# Seq2Seq Chatbot

- Python3

- [NumPy](https://www.numpy.org) 1.16.2
- [tqdm](https://github.com/tqdm/tqdm) 4.28.1
- [TensorFlow](https://github.com/tensorflow/tensorflow) 1.13.1
- [TensorLayer](https://github.com/zsdonghao/tensorlayer) 1.11.1
